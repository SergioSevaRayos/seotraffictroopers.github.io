Términos SEO Esenciales

Palabras que harán que entiendas mejor el SEO

Algoritmo

Conjunto de reglas que utilizan los motores de búsqueda para determinar el ranking de las páginas web.

Analítica web

Recopilación y análisis de datos sobre el tráfico y el comportamiento de los usuarios en un sitio web.

Anchor text

El texto visible de un enlace. Es importante porque le dice a Google de qué trata la página enlazada.

Anchor text diversity

Utilizar diferentes variaciones de texto ancla para los enlaces que apuntan a tu sitio web.

Arquitectura web

La estructura y organización de un sitio web, incluyendo la forma en que se enlazan las páginas entre sí.

Atributo ALT

Texto alternativo que se añade a las imágenes para que los motores de búsqueda entiendan de qué tratan.

Autoridad de dominio (DA)

Métrica que mide la fuerza y relevancia de un dominio web.

Backlink

Enlace que apunta a tu sitio web desde otro sitio web. Los backlinks son un factor importante para el SEO.

Black Hat SEO

Prácticas de SEO que intentan manipular los rankings de búsqueda de forma poco ética.

Bounce rate

Porcentaje de visitantes que abandonan un sitio web después de ver una sola página.

Broken links

Enlaces que apuntan a páginas que ya no existen.

Canonicalización

Proceso de indicar a los motores de búsqueda cuál es la versión preferida de una página web.

Canibalización de palabras clave

Ocurre cuando varias páginas de un mismo sitio web compiten por las mismas palabras clave.

Cloaking

Técnica de Black Hat SEO que consiste en mostrar contenido diferente a los motores de búsqueda y a los usuarios.

Content marketing

Creación y distribución de contenido valioso y relevante para atraer a tu público objetivo.

Contenido duplicado

Contenido que aparece en varias páginas web, ya sea dentro del mismo sitio o en sitios diferentes.

Contenido evergreen

Contenido que sigue siendo relevante y útil a lo largo del tiempo.

Conversion rate

Porcentaje de visitantes que completan una acción deseada en un sitio web.

Crawl budget

Es el número de páginas que un motor de búsqueda rastrea en un sitio web durante un período determinado.

CTR

Porcentaje de usuarios que hacen clic en un enlace.

Deep Links

Enlaces que apuntan a páginas específicas dentro de un sitio web.

Dwell Time

Tiempo que un usuario pasa en una página después de hacer clic en un resultado de búsqueda.

Enlazado interno

Enlaces que conectan diferentes páginas dentro de un mismo sitio web.

Exit Rate

Porcentaje de usuarios que abandonan un sitio web desde una página específica.

Experiencia de usuario (UX)

La calidad de la experiencia que tiene un usuario al interactuar con un sitio web.

Footer

La sección inferior de un sitio web que suele contener información de contacto, enlaces a páginas importantes y derechos de autor.

Freshness

Factor de ranking que premia el contenido nuevo o actualizado.

Google Analytics

Herramienta de Google que proporciona datos sobre el tráfico y el comportamiento de los usuarios en un sitio web.

Google Search Console

Herramienta de Google que permite a los webmasters monitorizar el rendimiento de su sitio web en los resultados de búsqueda.

Heading tags (H1-H6)

Etiquetas HTML que se utilizan para estructurar el contenido de una página web.

HTTP/2

Versión mejorada del protocolo HTTP que permite una carga más rápida de las páginas web.

Indexación

Proceso mediante el cual los motores de búsqueda rastrean y añaden páginas web a su índice.

Intención de búsqueda

El objetivo que tiene un usuario al realizar una búsqueda en un motor de búsqueda.

Keyword

Palabra o frase que los usuarios utilizan para buscar información en los motores de búsqueda.

Keyword research

Proceso de identificar las palabras clave que utilizan los usuarios para buscar información relacionada con tu negocio.

Keyword stuffing

Práctica de incluir una gran cantidad de palabras clave en el contenido de una página web de forma artificial.

Landing Page

Página diseñada específicamente para convertir visitantes en leads o clientes.

Latent Semantic Indexing (LSI)

Algoritmo que utiliza Google para entender el significado y la relación entre las palabras clave en un texto.

Leads

Personas que han mostrado interés en tu producto o servicio y han proporcionado sus datos de contacto.

Lighthouse

Herramienta de Google para auditar la calidad de una página web, incluyendo rendimiento, accesibilidad, SEO y buenas prácticas.

Link building

Estrategia para conseguir enlaces de calidad que apunten a tu sitio web.

Link juice

El valor que transmite un enlace de una página a otra.

Long tail keyword

Palabra clave de cola larga, es decir, una frase de búsqueda más específica y con menos volumen de búsquedas.

Meta descripción

Un breve resumen del contenido de una página web que aparece en los resultados de búsqueda.

Métricas de Engagement

Datos que miden cómo interactúan los usuarios con un sitio web, como el tiempo en la página, páginas por sesión y tasa de rebote.

Microdatos

Formato de marcado semántico que ayuda a los motores de búsqueda a entender el contenido de una página.

Mobile SEO

Optimización de un sitio web para que sea compatible con dispositivos móviles.

NoFollow

Atributo que se añade a un enlace para indicar a los motores de búsqueda que no deben seguir ese enlace.

Optimización Off-Page

Prácticas de SEO que se realizan fuera de un sitio web, como la construcción de enlaces.

Optimización On-Page

Prácticas de SEO que se realizan dentro de un sitio web para mejorar su ranking.

Outbound Links

Enlaces que apuntan desde tu sitio web hacia otros sitios. Deben ser relevantes y de alta calidad.

PageRank

Algoritmo de Google que mide la importancia de una página web en función de la cantidad y calidad de los enlaces que apuntan a ella.

Palabra clave principal

La palabra clave más importante para la que se quiere posicionar una página web.

Penalización de Google

Sanción que Google impone a un sitio web que no cumple con sus directrices.

Pogo-Sticking

Comportamiento del usuario en el que este hace clic en un resultado de búsqueda, pero regresa rápidamente a la SERP para elegir otro resultado. Puede indicar que el contenido no satisface la intención de búsqueda.

RankBrain

Sistema de inteligencia artificial de Google que ayuda a interpretar las consultas de búsqueda y a mejorar la relevancia de los resultados.

Rastreo (Crawling)

Proceso mediante el cual los motores de búsqueda envían robots (arañas) a rastrear la web y descubrir nuevas páginas.

Redirección 301

Redirección permanente de una URL a otra.

Responsive Design

Diseño web que se adapta automáticamente a diferentes dispositivos (móviles, tablets, escritorios), crucial para el SEO móvil.

Rich Snippets

Fragmentos enriquecidos que aparecen en los resultados de búsqueda, mostrando información adicional como calificaciones, imágenes o precios.

Roadmap (Hoja de ruta)

Es un plan estratégico que define los objetivos de SEO a largo plazo y las acciones necesarias para alcanzarlos. Un roadmap bien definido ayuda a priorizar tareas, asignar recursos y medir el progreso de la estrategia SEO.

Search Engine Results Page (SERP)

La página que muestra Google después de que un usuario realiza una búsqueda. La SERP incluye resultados orgánicos, anuncios pagados y otros elementos como mapas o imágenes.

SEO (Search Engine Optimization)

Conjunto de técnicas que se utilizan para mejorar el ranking de un sitio web en los resultados de búsqueda.

SEO local

Estrategia para optimizar la presencia online de un negocio local en los resultados de búsqueda. El SEO local es importante para atraer clientes que se encuentran en la misma zona geográfica que el negocio.

SEO técnico

Optimización de aspectos técnicos de un sitio web, como la velocidad de carga, la estructura de la URL y la indexación, para mejorar su rendimiento en los motores de búsqueda.

SERP (Search Engine Results Page)

La página de resultados que muestra un motor de búsqueda después de que un usuario realiza una búsqueda.

Siloing

Estrategia de organización del contenido en un sitio web en categorías temáticas para mejorar la estructura y la relevancia para los motores de búsqueda.

Sitemap

Archivo que contiene una lista de las páginas de un sitio web y que ayuda a los motores de búsqueda a rastrearlas e indexarlas.

Thin Content

Contenido de baja calidad o poco valor que no satisface las necesidades del usuario y puede ser penalizado por Google.

Time on site

Cantidad de tiempo que los visitantes pasan en un sitio web. Un mayor tiempo en el sitio puede indicar que el contenido es relevante y atractivo.

Title tag

El título de una página web que aparece en los resultados de búsqueda y en la pestaña del navegador.

Topical Authority

Autoridad que un sitio web tiene sobre un tema específico, basada en la profundidad y calidad del contenido que ofrece.

URL (Uniform Resource Locator)

La dirección web de una página.

User experience (UX)

La calidad de la experiencia que tiene un usuario al interactuar con un sitio web. Una buena experiencia de usuario es fundamental para el SEO, ya que Google premia a los sitios web que ofrecen una experiencia positiva a los usuarios.

Voice Search

Búsquedas realizadas a través de asistentes de voz como Google Assistant o Siri. Requiere un enfoque en palabras clave conversacionales.

Web Vitals

Métricas esenciales para medir la experiencia del usuario en una página web, incluyendo Largest Contentful Paint (LCP), First Input Delay (FID) y Cumulative Layout Shift (CLS).

White Hat SEO

Prácticas de SEO que siguen las directrices de los motores de búsqueda y que se centran en ofrecer contenido de calidad y una buena experiencia de usuario.

Zero-Click Content

Contenido diseñado para responder directamente a las consultas de los usuarios en los resultados de búsqueda, como los Featured Snippets.
